A PASSION REKINDLED

I started coding in the early 80s, when at the age of 10, I taught myself BASIC using an old Radio Shack TRS-80 and a reference manual. In later years I occassionally dabbled with programming, learning bits of HTML and C here and there. But it was because of the COVID-19 pandemic that I reconnected with coding in a serious way. During the isolation, I learned HTML/CSS/JavaScript, Python, the basics of Java and Lua, and how to use Git and GitHub. More recently I have taken up C++, the PyQT/Pyside UI package, and Ruby on Rails.


I have always been fascinated by complex, abstract structures. Excelling in math even as a toddler, I completed an honours undergraduate degree in mathematics and philsosophy before veering into law. While I plan to continue coding as a hobby regardless, I am excited about seeking a way to fold mathematical analysis or computer programming into my future career path.
